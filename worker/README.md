# agenvoy-kura

Static landing page for **KuraDB** on Cloudflare Workers.

Generated by [Claude Code](https://claude.ai/claude-code).

## Develop

```bash
npm install
npm run dev      # wrangler dev — serves ./public locally
```

## Deploy

```bash
npm run deploy   # wrangler deploy — publishes worker "agenvoy-kura"
```

The site is pure static assets under `public/` (single-file `index.html` + brand SVGs), served via the Workers `[assets]` binding. No build step.

## Custom domain

After the first deploy, bind `kura.agenvoy.com` to the `agenvoy-kura` worker in the Cloudflare dashboard (Workers → agenvoy-kura → Settings → Domains & Routes), or add a `[[routes]]` block to `wrangler.toml`.
